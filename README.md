# Finanz-OS: Automatisierter Bank-Tracker 🚀

Ein intelligentes System zur Kategorisierung und Nachverfolgung von Bankumsätzen. 
Kombiniert die Power von **Python (Pandas & Logik)** mit der Benutzerfreundlichkeit von **Excel (VBA)**.

## ✨ Features
- **Automatischer Import:** Lädt CSV-Exporte deiner Bank direkt in Excel.
- **Smarte Kategorisierung:** Python-Skripte ordnen Umsätze basierend auf Vorlagen (template.json) zu.
- **Template-System:** Erstelle neue Regeln direkt in Excel, die sofort in das Python-Gedächtnis übernommen werden.
- **Workflow-Schutz:** Verhindert Duplikate durch strikte Validierung (alles muss "grün" sein).
